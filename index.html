Sissejuhatus

Mis on HTML?

Hüperteksti märgistuskeel, standardiseeritud süsteem tekstifailide sildistamiseks, et saavutada veebilehtedel fondi-, värvi-, graafika- ja hüperlingiefekte.

Mis on CSS?

CSS on küljendamisel kasutatav märgistuskeel, milles märgitakse üles peamiselt veebilehtede kujundust.

HTML tähendus

HTML on lühend ingliskeelsest terminist Hyper Text Markup Language. HTML on veebilehtede loomise standardne märgistuskeel. HTML kirjeldab veebilehe struktuuri. HTML koosneb elementide seeriast. HTML-elemendid ütlevad brauserile, kuidas sisu kuvada.

Kasulikud kohad

HTML faili seletus

1. Struktuur

DOCTYPE HTML -> ütleb brauserile, et tegemist on HTML dokumendiga

html lang="et" -> lang="et" näitab, et leht on eesti keeles

head -> sisaldab andmeid, mida külastaja ei näe

body -> kogu nähtav sisu, mida ka kasutaja näaeb

2. Meta ja lingid

Meta charset="UTF-8" -> määrab tähemärgikodeeringu (UTF-8, et täpitähed töötaksid)

meta name="viewport" content="width=device-width, initial-scale=1.0" -> muutab lehe mobiilisõbralikuks

title -> veebilehe vahekaardi pealkiri

link rel="..." href="...css -> ühendab CSS-faili

3. Sisu ja paigaldus

div class=hero-img" -> suur taustapilt esilehel

div class=hero-text -> tekst hero pildi sees

h1 -> suurim pealkiri

h2 -> teise tasemel pealkiri (alapealkiri)

main -> peasisu

sectipn id="..." -> sisublokk, millel on kindel ID

p -> kõik tekstid

em -> kursiivne tekst

u -> alla joonitud tekst

4. Navigatsioon ja lingid

nav aria-label="..." -> navigeerimisriba

ul class="nav" -> loetelu linkidega

li,a href="#sissejuhatus" loendi element lingiga, href="#id viib sama lehe sees õigesse kohta

a href"..." target="blank" rel="noopener noreferrer" -> link teisele veebilehele, target="blank avab lingi uues aknas ja rel on turvalisuse pärast

5. Meedia

video controls width="640" height="360" poster="..." -> video element, poster on pilt, mis kuvatakse enne video käivitamist ja controls lisab mängimise nupud

source src"..." type="video/mp4 -> video allikas

img src="..." alt="..." class"..." -> pilt, kus src on faili asukoht, alt alternatiivne tekst, class CSS-i jaoks stiilide sidumine

CSS faili seletus

1. Mõõdud ja paigutus

width/height -> määrab elemendi laiuse ja kõrguse

margin -> välisvahe teiste elementidega

padding -> sisemine vahe elemendi serva ja sisu vahel

float: left -> paneb elemendi liikumna teksti kõrvale vasakule

display: flex -> paigutab elemendid reas või veerus paindlikult

justify-content: centre -> seab flex-elemendid horisontaalselt keskele

gap -> vahe flex või grid elementide vahel

position: relative -> alus positsioneerimiseks

position: absulute -> paigutab elemdi täpselt kordinaatitega

transform: translate(-50%, -50%) -> nihutab elemendi täpselt keskele

2. Värvid ja taust

color -> teksti värv

background-color -> taustavärv

vackground-image -> lisab taustapildi

background-position -> määrab, kust taustapilt algab

background-repeat: no-repeat -> keelab taustapildi kordumise

background-size: cover -> venitab taustapildi nii, et see katab kogu ala

linear-gradient -> värvide üleminek sirgjooniliselt

radial-gradient -> värvide üleminek ringikujuliselt

3. Tekst ja font

font-family -> määrab kasutava kirjatüübi

font-size -> teksti suurus

font-weight: bold -> paks tekst

line-height -> lõikude rea kõrgus

text-align: centre -> tekst keskele

text-decoration: underline -> alla joonitud tekst

clamp(min, soovituslik, max) -> automaatselt kohanduv font või mõõt vastavalt ekraani suurusele

4. Pildid ja meedia

border-radius -> ümarad nurgad

float -> paneb pildi teksti kõrvale

.gif {width: ..., height: ..., border-radius: ... } -> määrab GIF-i suuruse ja ümardab nurgad

5. Lingid ja menüüd

list-style: none -> eemaldab loendi täpid

text-decoration: none -> eemaldab lingi allajoonimise

transition: color o.3s ease -> teeb värvimuutuse sujuvaks

:hover -> stiil, mis rakendub, kui hiir liigub elemendi peale

:active -> stiil, kui elementi vajutatakse

6. Ekstra

@font-face {...} -> lubab lisada omaenda fondifailis projekti kaustast

vh/vw -> mõõtühikud, mis võtavad aluseks ekraani kõrguse või laiuse protsendi

rem/em -> mõõtühikud, mis sõltuvad fontide suurusest
